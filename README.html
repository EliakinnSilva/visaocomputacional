Passo a Passo para Instalação de Bibliotecas Python

Passo 1: Abrir o Terminal/Prompt de Comando

Dependendo do seu sistema operacional, você vai abrir o terminal (no macOS ou Linux) ou o prompt de comando (no Windows).

Passo 2: Criar e Ativar um Ambiente Virtual (Opcional, mas Recomendado)

É uma boa prática criar um ambiente virtual para seu projeto. Isso ajuda a gerenciar as dependências sem afetar outros projetos.

Criar um Ambiente Virtual:

python -m venv meu_ambiente

Substitua meu_ambiente pelo nome que você deseja dar ao seu ambiente virtual.

Ativar o Ambiente Virtual:

No Windows:

meu_ambiente\Scripts\activate

No macOS ou Linux:

source meu_ambiente/bin/activate
Passo 3: Instalar as Bibliotecas

Agora, você pode instalar as bibliotecas necessárias:

Instalar OpenCV e Cabeçalho:

pip install opencv-python-headless

Instalar NumPy, TensorFlow, Pillow e SciPy:

pip install numpy tensorflow pillow scipy
Passo 4: Verificar as Instalações

Para garantir que tudo foi instalado corretamente, você pode rodar o seguinte comando para listar as bibliotecas instaladas:

pip list

Isso deve mostrar todas as bibliotecas que você instalou, incluindo opencv-python-headless, numpy, tensorflow, pillow e scipy.

Passo 5: Desativar o Ambiente Virtual (quando terminar de trabalhar)

Para desativar o ambiente virtual, basta rodar:

deactivate

Resumo dos Comandos

# Criar um ambiente virtual (opcional)
python -m venv meu_ambiente

# Ativar o ambiente virtual
# No Windows:
meu_ambiente\Scripts\activate
# No macOS ou Linux:
source meu_ambiente/bin/activate

# Instalar bibliotecas
pip install opencv-python-headless
pip install numpy tensorflow pillow scipy

# Verificar as instalações
pip list

# Desativar o ambiente virtual (quando terminar)
deactivate
